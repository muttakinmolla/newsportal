1. What is the difference between var , let , const ?

Answer : The scope of a var variable is functional scope and it can be updated and re-declared into the scope. The scope of a let variable is block it can be updated but can not re-declared. On the other hand The const variable is block scope. It can not be re-updated and re-declared.

3. Why we use template string?

The big advantage of template string is when it is necessary to insert variables into a string phrase. The other advantage of template string in not needing to escape double quote or single quote characters using a backslash. It easy to write multiline code.
